# Task 6: House Price Prediction

## Objective
Predict house prices based on property features like Area, Bedrooms, Bathrooms, Floors, and Garage.

## Dataset
House price dataset with columns: Id, Area, Bedrooms, Bathrooms, Floors, YearBuilt, Location, Condition, Garage, Price.

## Model
- Linear Regression

## Results
- MAE: 244,425
- RMSE: 280,564
- Scatter plot shows actual vs predicted prices.

## Insights
The model predicts reasonably well, but can be improved with more features or advanced models.
